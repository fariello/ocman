# whatnext

Read-only surveyor: answer "what should I work on next here?" by surveying the repo's
plans, staged prompts, comms inbox (headers only), and TODO, then returning a prioritized,
reasoned recommendation. Recommends, never acts. Run `/whatnext` (optionally with a focus
like `/whatnext release`), or from any agent: "read and execute
`.agents/workflows/whatnext/whatnext.md`". See `.agents/workflows/index.md` for the full catalog.
