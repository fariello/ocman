# Walkthrough - IPD 04 - Compact Secret Visibility, Expunge, and Yes Option

Implemented matched secret display (masked and raw context), outbound transmission redaction (expunging), prompt-triggered file rewriting on disk, and a `-y`/`--yes` bypass flag.

## Changes Made

### 1. Secret Scanning & Matching
- Updated `SecretHit` dataclass in [ocman.py](file:///home/gfariello/VC/ocman/ocman.py) to include match span fields `col_start` and `col_end`.
- Rewrote `scan_for_secrets` to use `re.Pattern.finditer` rather than `search`, enabling exact span tracking for all matched secrets.

### 2. Redaction & Masking Helpers
- Implemented `redact_secrets(text, hits)` to merge overlapping/adjacent secret spans per line, replacing them right-to-left with safe placeholders like `<REDACTED:safe_kind>`. Placeholders are sanitized to prevent re-triggering keywords.
- Implemented `mask_line(line, hits)` to replace match spans on a line with asterisks (`*`) for masked display.
- Implemented `_display_secrets_context(text, hits, mode, is_tty)` to print matched lines. Raw reveal is protected behind a TTY confirm ("type 'reveal'").

### 3. Egress Guards Refactoring
- Updated `check_egress_guards` signature to support `expunge_secrets`, `show_secrets`, and `interactive`.
- If secrets are flagged and no skip flags are passed, interactive review triggers a prompt offering `[s]how masked / [r]eveal raw / [e]xpunge / [a]bort`.
- Redacted text is validated to verify it contains zero matches on a second scan.

### 4. Integration with Compact & Filter
- Added `--show-secrets` and `--expunge-secrets` options to `compact` and `filter` command subparsers.
- Handled mutual exclusivity constraints (raising error if both `--allow-secrets` and `--expunge-secrets` are specified).
- Updated `run_compaction` and `cli_filter` to pass flags and return whether content was expunged.
- If compaction content is expunged, `main()` prompts (or auto-applies with `-y`) to rewrite all written on-disk recovery files via `redact_secrets()`.
- Added `-y`/`--yes` support to `compact` to bypass confirmation and cost estimates prompts.

### 5. Tests & Verification
- Added new test cases in [tests/test_ocman.py](file:///home/gfariello/VC/ocman/tests/test_ocman.py) verifying scan & redaction, overlapping span merges, masked outputs, and CLI options verification.
- Verified all 121 tests pass successfully.
